O CSS documento de estilização em cascata criado em 1993, com sua primeira versão amplamente utilizada em 1997. A linguagem permite definir regras que definem caracteristicas para os elmentos HTML como cor, borda e comportamentos, além de animações e transições.

Ao longo dos anos o css evoluiu e hoje temos o CSS 1 2 e 3, cada uma das versões adicionou novos recursos que eram demandados no periodo.

  1. CSS 1 1997 - própriedades básicas de estilização e formatação, além de controle do posicionamento dos elementos html com propriedades float e position
  2. CSS 2 1980 - adição de propriedades de animação e transição e melhoria do position
  3. CSS 3 - incremento gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento com display:flex (2008) e grid (2013)

Uma regra css pode ser definida com a seguinte sintaxe

Seletor{
    Propriedade1:Valor;
    Propriedade2:Valor;

    PropriedadeN:Valor;
}

Podemos incluir código CSS de 3 formasw diferentes

CSS externo
Quando definimos as regras CSS a serem utilizadas por meio de um arquivo externo .css, vinculando ele ao html por meio da tag < link > no head
CSS incorporando "no documento"
Quando definimos as regras de css por meio da tag <style >
CSS inline
Quando definimos código de css de estilização no próprio elemento HTML como: Texto de cor vermelha com CSS inlide

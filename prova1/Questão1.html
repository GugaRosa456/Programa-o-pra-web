O processo de acesso ocorre entre um servidor e um cliente, o cliente pede um solicitação e o servidor manda uma resposta.

o servidor manda uma copia da página web para o cliente

Quando o cliente coloca a URL na pesquisa, o DNS( uma especie de catalogo de endereços) que redimenciona a página para onde o cliente quer

o TCP faz gerencia os dados entre o cliente e o servidor

HTTP cria a linguagem para a comunicação entre o servidor e o cliente

todos esse processo faz com que o servidor e o cliente se comuniquem para um bom desempenho

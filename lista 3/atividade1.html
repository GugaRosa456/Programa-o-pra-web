Grid Garden

No jogo Grid Garden, na fase 21, devemos usar um comando em especifico para achatar a coluna e assim colocar água apenas no lugar em que a cenoura está.

O codigo que usamos é grid-template-columns, o grid é utilizados para organizar o elementos em linhas e colunas, o columns é apenas para selecionar os elementos da coluna.

a porcentagem é para determinar quanto espaço vai ter de coluna, no caso, qual a porcentagem de agua será utiliado para regar apenas o espaço que está a cenoura.
